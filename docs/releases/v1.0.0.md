# SwiftMessengerLab 1.0.0

首个稳定版把一个完全本地的 Messenger 发送链路和一个零基础 Swift/UIKit 类型实验室放在同一 App 中。

## 本版包含

- Messenger：乐观插入、`queued → sending → sent/failed`、同 id 重试与 JSON 缓存。
- Learn：20 节短课、52 张全局唯一类型卡、语言概念、类型/API 搜索。
- Practice：每张类型卡都能进入 App、LLDB、源码三层实验；Reset 与 Messenger 数据隔离。
- Compiler：5 个小于等于 20 行的样本，生成 SILGen、SIL、LLVM IR 和 ARM64 片段。
- Evidence：9 个 Core、9 个 UI 场景，以及 compiler/public/showcase 自动门禁。
- Showcase：GitHub Pages、原创图标和三张真实 Simulator 截图。

## 运行

```bash
git clone https://github.com/estelledc/SwiftMessengerLab.git
cd SwiftMessengerLab
make build
make run
```

需要 macOS、Xcode 16+ 和 iOS 17+ Simulator。

## 边界

本版本只包含公开 API、虚构数据和原创实现；不连接真实聊天服务，不复制商业客户端实现，不提供 IPA。账号、附件、推送、加密、SwiftUI 与 Objective-C 混编不属于 1.0.0。
